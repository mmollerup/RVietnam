General idea

The general idea behind linear regression is that a relationship between a variable of interest, say \(y\), can be expressed as a polynomial function of one or more explanatory variables \(x_1, \dots, x_k\):

\[y = a + b_1 \cdot x + b_2 \cdot x^2 + \dots + b_n \cdot x^n \]

0r

\[y = a + b_{1,1} \cdot x_1 + b_{1,2} \cdot x_1^2 + \dots + b_{1,n} \cdot x_1^n + \\ b_{2,1} \cdot x_2 + b_{2,2} \cdot x_2^2 + \dots + b_{2,m} \cdot x_2^m \\ + \dots + \\ b_{k,1} \cdot x_k + b_{k,2} \cdot x_k^2 + \dots + b_{k,l} \cdot x_k^l\]

These models are very easily interpretable and we make them automatically all the, e.g.when we see a child, we infer the age by estimating the height (e.g.a child of 110 cm is probably 5 or 6 years old).

Linear regression has been around for hundred of years and are still some of the most used algorithms around.

There are more advanced algorithms out there, like random forests or artificial neural networks, but linear regression is still serviceable for a large number of uses.

The lm() and glm() functions

If we want to fit a linear regression model in R, we need to use either the lm() or the glm() function.

The lm() function fits a linear regression model, whereas the the glm() function can handle logistic and poisson regression, too. If used to fit linear regression models, they will produce the same results.

model1 <- lm(mpg ~ hp, data = mtcars)

A note on reducing the number of explanatory variables I

If you specify your model with too many explanatory variables, we say that it is over-fitted.

This may be an issue because the model fits your data too well, but doesnt generalise to other data. After all, part of the reason for modelling data is to be able to predict values for unpaired variables, so we seek a balance between explanatory power and generalisability.

In order to include an explanatory variable (or predictor) in your model, there should be a logical reason why it would have explanatory power, and it should be significant according to the model summary.

A note on reducing the number of explanatory variables II

Another way of doing it is the following:

  • Start with the model that contains all potential predictors
  • Remove one predictor at a time
  • Keep this model if it improves the measure of accuracy (adjusted R-squared)
  • Iterate until no further improvement is made.
